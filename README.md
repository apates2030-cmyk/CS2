# Project Title Task Organizer for Academics
---
## Overview/Description
A task organizer that helps students track their homework or assignments for the day.
---

## Features
The project can:
-Add tasks
-View tasks
-Mark tasks as completed
---

## How to Run the Program
1. Have Python installed.
2. Download the file `task_org.py`.
3. Open a command prompt.
4. Run the program by clicking `Run..`
5. Follow the on-screen instruction to enter tasks.
---
## Logic Plan
(Start the Program)
        |
        v
[Empty Task List]
        |
        v
"Decision":
1. Add a Task
2. View Tasks
3. Mark a Task as Complete
4. End
        |
        v
|
> If choice 1 -> [Input the Task Description] -> [Add to the Task List] -> Back to "Decision"
---
|
> If choice 2 -> [Print Tasks on the List] -> Back to "Decision"
---
|
> If choice 3 -> [Input the Task Number] -> [Mark as Complete] -> Back to "Decision"
---
|
> If choice 4 -> (End Program)

---
